## 🌍 About BeLocal

BeLocal is a revolutionary platform connecting tourists with authentic local experiences across Uzbekistan. Skip the tourist traps and live like a local with certified guides, real food, and genuine cultural connections.

### Why BeLocal?

✅ **Authentic Experiences** - Real local life, not staged tourism
✅ **Budget-Friendly** - Explore without breaking the bank
✅ **Support Locals** - Help certified guides earn extra income
✅ **Community Reviews** - Honest feedback from real travelers
✅ **Cultural Learning** - Traditions, customs, and language from natives
✅ **Verified Content** - Professional quality control on all content

## 🏙️ Our Cities

1. **Samarqand** 🕌 - Heart of historic and ancient buildings
2. **Bukhara** 🏛️ - Ancient trading hub
3. **Khiva** 🏜️ - Perfectly preserved walled city
4. **Tashkent** 🌆 - Modern capital city
5. **Shahrisabz** ⛰️ - Birthplace of Amir Timur
6. **Urgench** 🌊 - Gateway to Khorezm

- **Package Manager**: npm

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Bektosheva/belocal.git

# Navigate to project
cd belocal

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
